Kickoff: Uppsala Azure User Group

📅 May 7, 2026 🕕 17:30 – 20:00 📍 Stationsgatan 23, Uppsala

Welcome to the very first meetup of Uppsala Azure User Group! Join us for two great sessions on DevOps and Azure, followed by networking.

Speakers

  • Björn Sundling – Microsoft MVP
  • Sebastian Claesson – TechLead @ Viedoc
  • 17:30 Opening & pre-mingle with food & beverages
  • 18:00 Björn Sundling – Dark Magic of PowerShell and other dark things

    Björn will summon the forbidden arts of PowerShell — the scripts your sysadmin warned you about, the one-liners that make security teams nervous, and the automation tricks that probably void your warranty. Viewer discretion advised.

  • 18:45 Sebastian Claesson – Maester: Azure DevOps Security

    Maester is a PowerShell-based security test automation framework that continuously monitors and validates the security configuration of your Microsoft 365 and Entra ID environment. Sebastian will cover the new feature for checking security posture in Azure DevOps.

  • 19:30 Q&A & networking
  • 20:00 Closing

About the Group

Uppsala Azure User Group is a local community for anyone interested in Microsoft Azure — from beginners to seasoned cloud architects. We meet regularly to share knowledge, learn from each other, and stay up to date with the Azure ecosystem.


Whether you're a developer, IT pro, student, or just cloud-curious, you're welcome to join us. Our events are free and open to everyone.
